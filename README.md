# ☕ Coffee Shop Website

A modern and responsive Coffee Shop Website built using HTML, CSS, and JavaScript. This project showcases a stylish coffee shop landing page with interactive sections, mobile-friendly navigation, testimonials slider, gallery, and contact form.

## 🚀 Live Demo

Live website link here:

https://hetal22kar125.github.io/coffee-shop-website/

---

## 📌 Features

- Responsive Design
- Modern Coffee Shop UI
- Mobile Navigation Menu
- Hero Section
- About Us Section
- Menu Showcase
- Customer Testimonials Slider
- Image Gallery
- Contact Section
- Smooth User Experience
- Swiper.js Integration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Swiper.js

## 🎯 Sections Included
